# Comprehensive Annual and Monthly Trading Data Analysis for Financial Market Insights

## Overview
This repository contains a Python-based implementation to analyze and visualize trading data from the Abu Dhabi Stock Exchange for the year 2026. It includes functionalities to process, analyze, and visualize annual and monthly trading data segmented by investor type and sector.

## Features
1. **Interactive Dashboards:** Explore data via customizable charts and graphs.
2. **Advanced Filtering Options:** Filter data by investor type, sector, and time period.
3. **Downloadable Reports:** Export data in CSV and XLSX formats for offline use.
4. **Predictive Analytics:** Predict future trading trends using historical data.
5. **Custom Notifications:** Set up alerts for significant changes in trading patterns.

## Prerequisites
To run the code in this repository, you need:
- Python 3.6+
- Pandas
- Matplotlib

Install the required libraries using pip:
bash
pip install pandas matplotlib


## How to Use
1. Clone the repository:
bash
git clone https://github.com/yourusername/trading-data-analysis.git


2. Navigate to the project directory:
bash
cd trading-data-analysis


3. Place your datasets (`Annual_Trading_Summary_by_Investor_Type_2026.csv` and `Monthly_Trading_Summaries_by_Sector_2026.csv`) in the `data/` folder.

4. Run the Python script:
bash
python analysis.py


5. View the generated visualizations and analyze the results.

## Example Usage
The provided script includes two example analyses:
1. **Annual Buy vs Sell Value by Investor Type:** Visualize the comparison of buy and sell values for different investor types in 2026.
2. **Monthly Buy Value by Sector:** Analyze the total buy value for each sector across all months of 2026.
